# STA 6714 Final Project – Data Preparation

## Project Overview

This project focuses on **data cleaning, preprocessing, and exploratory analysis** as part of the STA 6714 final project. The purpose of this notebook is to transform raw data into an analysis-ready format suitable for statistical modeling and inference.

The notebook demonstrates a clear, reproducible workflow commonly used in data science and applied statistics projects.

## Contents

* Data loading and initial inspection
* Data cleaning and validation
* Handling missing and inconsistent values
* Feature engineering and variable transformation
* Exploratory data analysis (EDA)
* Summary statistics and visualizations

## Files in This Repository

* `STA6714-Final_Project-DataPrep.ipynb` — Main Jupyter notebook containing all data preparation and analysis steps
* `README.md` — Project documentation
